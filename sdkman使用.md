安装

```plain
curl -s "https://get.sdkman.io" | bash

source "$HOME/.sdkman/bin/sdkman-init.sh"

sdk version
```



列出所有支持的软件包

sdk list



sdk list java



sdk install java ${version}



sdk uninstall java ${version}



sdk use java version  (当前窗口生效)



sdk default java version



sdk current java
